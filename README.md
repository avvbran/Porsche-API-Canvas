# Porsche 911  - API Canvas 2D

Aplicación web desarrollada con HTML, CSS, Bootstrap y JavaScript.

## Descripción
En este proyecto se utiliza la API Canvas 2D para dibujar un Porsche 911 inspirado en una imagen real, utilizando figuras básicas.

## Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Canvas API 2D

## Características
- Dibujo generado por código
- Imagen de referencia incluida
- Figuras básicas
- Código organizado
- Navbar
- Favicon personalizado
